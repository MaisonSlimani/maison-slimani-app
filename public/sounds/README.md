# Fichiers audio

Ce dossier contient les fichiers audio utilisés pour les effets sonores du site.

## Fichiers requis

- `add-to-cart.mp3` - Son d'ajout au panier

## Note

Si ces fichiers n'existent pas, le site fonctionnera normalement sans les effets sonores. Les erreurs 404 sont gérées silencieusement.
